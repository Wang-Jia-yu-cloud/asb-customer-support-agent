# ASB Customer Support Agent (RAG)

🔗 **Live Demo:** [asb-customer-support-agent.streamlit.app](https://asb-customer-support-agent.streamlit.app)

---

## 🚀 What it is

An AI-powered banking support chatbot that answers customer queries using a Retrieval-Augmented Generation (RAG) pipeline built on real ASB Help data.

## 💼 Why it matters

- Automates repetitive customer support queries
- Provides instant, 24/7 responses
- Demonstrates real-world application of LLM + RAG in banking

## 🧠 How it works
```
User Query → Intent Detection → Query Rewriting → Vector Search (Pinecone) → LLM Response
```

## ⚙️ Key Highlights

- Built on 1,000+ real ASB help articles
- Supports follow-up questions (conversation memory)
- Handles ambiguous queries with clarification
- Deployed with live Streamlit interface

## 🛠 Tech Stack

Python · Pinecone · OpenAI (GPT-4o-mini) · Streamlit
